Демо Log-loss

Бинарная классификация на плоскости

Два класса точек на плоскости признаков (x₁, x₂). Модель: P(y=1) = σ(w₁x₁ + w₂x₂ + b). Обучение — градиентный спуск по log-loss (кросс-энтропия). Оранжевая линия — граница решения w₁x₁ + w₂x₂ + b = 0. Подробности — на странице Log-loss (кросс-энтропия).

Данные и граница

ЛКМ — добавить точку выбранного класса. ПКМ — удалить ближайшую точку.
